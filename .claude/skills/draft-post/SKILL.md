# draft-post

Цель: написать пост в голосе (стиле) сущности NAME и сохранить как черновик. Публикация в группу = обычный `post`/`schedule`.

Если voice.md пустой/скаффолд — предложи /study-entity или скажи, что стиль ещё не выведен (см. CLAUDE.md). Конкретно: наполни профиль из `channels/NAME/<@handle>/posts/` или скажи, что стиль не выведен.

## Шаги

1. **Проверь гард изучения.** Гард изучения (см. CLAUDE.md): post/schedule/poll create/group reply требуют studied: true; если false — предложи /study-entity и остановись.

2. **Загрузи голос и память.** Прочитай:
   - `channels/NAME/skills/voice.md` — профиль голоса (тон, лексика, структура, форматирование);
   - `channels/NAME/CLAUDE.md` — память «мозга» сущности (контекст, ограничения, выученные уроки).

3. **Определи тему.** Если пользователь назвал тему сам — она приоритетнее плана. Иначе возьми ближайший незакрытый слот из `channels/NAME/content-plan.md`: незакрытый слот = строка плана со `status: planned` (не `scheduled` и не `posted`).

4. **Создай черновик.** Запусти:
   ```
   uv run tgcockpit drafts new --channel NAME --title "Заголовок" [--pillar P] [--image PATH ...]
   ```
   `--pillar` — из `pillars` сущности; `--image` повторяй для каждой картинки (до 10).

5. **Напиши тело.** Открой созданный файл `channels/NAME/drafts/<date>-<slug>.md` и впиши тело поста по voice.md. Формат текста — по /format-tg (посты — HTML руками; в чат можно markdown).
   Заполни frontmatter:
   - `title` — заголовок;
   - `pillar` — если у сущности есть `pillars` в `config.yaml` и тема в один из них ложится;
   - `status: draft`;
   - `schedule_at` — только если пост планируется на конкретное время позже.

6. **Покажи черновик** пользователю (заголовок + тело). НЕ запускай публикацию автоматически. Команда встанет в очередь на кнопку — скажи пользователю, что готово и ждёт подтверждения; не утверждай, что опубликовано (см. CLAUDE.md).

## Дальше

- Опубликовать сейчас: `uv run tgcockpit post --channel NAME --draft <date>-<slug>.md --now`
- Поставить в недельную очередь: `/schedule-week`
